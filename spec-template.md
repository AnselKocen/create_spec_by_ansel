### 背景系统

> 背景不是装饰，是氛围的基础。

**固定背景图**：`{{BACKGROUND_IMAGE_URL}}`

背景图特征（开发者据此适配 UI）：
- **色调**：{{COLOR_TONE}}
- **内容**：{{BG_CONTENT}}
- **氛围元素**：{{BG_AMBIENT}}

实现要求：
- `background-image` 固定铺满视口，`background-size: cover; background-position: center; background-attachment: fixed`
- 背景图上方叠加一层半透明暗色遮罩（`{{OVERLAY_COLOR}}` 左右），防止背景图案干扰前景内容的可读性。遮罩透明度可根据当前页面内容密度微调
- **可选增强**：在遮罩之上，可叠加 1-2 个极低透明度的 radial-gradient 色块做缓慢呼吸动画（周期 ≥ 15s）

> 原则：背景图是固定的氛围底图，所有动态效果和注意力焦点在前景的内容和 UI 上。

### 动效标准

- 所有 UI 过渡时长 200-800ms
- 使用 ease 或 cubic-bezier 缓动曲线，禁止线性运动
- 背景动画周期 ≥ 15s，极缓慢
- 尊重用户的 `prefers-reduced-motion` 系统偏好
