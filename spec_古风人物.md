- 所有过渡时长控制在 200-800ms
- 使用 ease 或 cubic-bezier 缓动曲线，禁止线性运动

- 动效必须有目的（提供反馈 / 引导注意），禁止为了装饰而加动画

## 十、核心机制约束

> 以下是**不可更改的游戏规则**，CC 开发时必须遵守。

### 历史准确性

- 称谓必须符合当时——不能用后世追封的名号
- 人物关系必须真实——谁和谁是什么关系必须基于真实历史
- 对话必须符合身份——皇帝、大臣、家人说话语气各不同
- 时间线不能错乱——事件先后顺序、人物年龄与史实一致

### 选择不影响剧情

- 故事永远按历史发展，玩家选什么都不改变走向
- 对话选项：2 个选项方向相近，只有语气差异，纯沉浸体验
- 决策选项：3 个选项都"说得通"，选后只看到反馈，不改变后续

### 知识来源

- AI 自身的历史知识（第一优先）+ 用户提供的 skills（第二优先）
- skills 往往只覆盖人物生平的一部分，AI 必须用自己的知识补全完整人生

### 叙事规则

- 全程第二人称「你」
- 文白混搭——日常白话亲切易读，关键处引用原文
- 生僻词紧跟括号注释（10 字以内）
- 每段 100-200 字，要有**具体的人名、地名、动作、感官细节**
- 写具体发生了什么，不写概述。反例："你经历了很多"；正例："大雪封路，你裹着单衣在驿站等了三天"
- 不要只写大事件——也要写**日常**：人物的生活细节、心理活动、与身边人的相处
- 要写出**感官**：天气、声音、气味、场景的画面感
- 事件之间要有**自然过渡**：不要从一个大事件直接跳到另一个，要写出期间的日常、心理变化、氛围转换
- **章内场景必须按时间顺序推进，先发生的事先讲，后发生的事后讲，禁止倒叙或时间跳跃**
- **每章的叙事围绕一件事循序渐进：先交代前因和处境，再通过对话展开事件发展，最后推向决策和章末高潮。禁止开章直接进入关键冲突，要让玩家先知道"事情是怎么走到这一步的"**
- 支持 `\n` 换行分段

### 对话机制

- speaker 必须是主角以外的人物
- NPC 台词有个性，像真人说话——有情绪、有犹豫、有脾气
- 回应固定 2 个，方向相近，只有语气差异
- followup 不影响走向

### 决策与反馈

- 恰好 3 个选项，`is_authentic: true` 只有 1 个，分值 3/1/0
- 3 个选项都应该让玩家觉得"说得通"，不要有明显的"正确答案"
- **选项文字中禁止出现暗示后果的语句**（如"这会导致……""这可能引发……"）——选择不影响剧情
- `context` 要充分交代背景和矛盾，让玩家理解为什么这个选择很难
- **反馈核心原则：故事永远只有一条线——真实历史。玩家的选择不会、也不能改变任何事。反馈的唯一目的是：告诉玩家历史上真正发生了什么，引发思考**

**与人物选择相同时：**
告诉玩家你们想到一起了，然后展开这个选择背后的具体经过、代价和后果

**与人物选择不同时（最重要的反馈类型），必须包含两部分，缺一不可：**
1. **简短理解玩家**（1 句）：你这样选，或许是因为……
2. **揭示人物真实选择**（核心，占 70% 篇幅）：但历史上的人物选了另一条路——[具体做了什么、说了什么、结果如何]

**反馈绝对禁止：**
- 描写"玩家的选择导致了什么后果"（玩家的选择没有后果，不要编造一个虚假的后果场景）
- "你错过了最好的时机"（玩家没有错过任何东西）
- "你在某某心中的分量轻了"（选择不改变任何事）
- "效果远不如……"（不存在效果对比）
- 只写玩家选择的展开，不提人物真实选择（这是最常见的错误！）
- 任何暗示"选错了""结局不同""代价变了"的措辞

**反馈正确示范（与人物不同时）：**

> 你选择了以退为进，用低姿态来打动他——这是很多人面对权力时的直觉。但历史上的她走了一条更决绝的路：她直接向高宗提出了废后的暗示，用的不是恳求，而是对王皇后「行厌胜之术」的指控。在那个后宫即战场的年代，退让从来不是她的选项。

> 你选择了沉默不言，等待更好的时机。但人物没有等——他当庭递上了那篇震动朝野的奏章，明知会触怒天子，仍然一字不改。「一封朝奏九重天，夕贬潮州路八千」，这不是冲动，而是他想了很久之后的决定。

**反馈正确示范（与人物相同时）：**

> 你和人物想到了一起。他/她确实选择了[具体做法]，[展开这个选择的经过和后续影响]。有时候，[一句基于史实自然引出的感悟]。

**反馈全局要求：**
- 每条 100-200 字
- **每条「与人物不同」的反馈，都必须明确写出人物真实的选择是什么**——这是反馈存在的意义
- 所有反馈的**开头和结构各不相同**——禁止模板化
- 每条必须包含至少一个**具体的历史事实**（人名、事件、时间、引文）
- 尽量引用人物原话（用「」包裹）
- **永远不描写玩家选择的"后果"**——玩家的选择没有后果，只有历史才有后果
- 语气温和平等——像朋友聊历史，不是老师判卷

### 章末感悟

- 每章末尾必须有 1 个 ending 场景
- 诗句优先选人物自己的作品
- annotation 必填：注明出处和白话释义
- wisdom（60-100 字）：带感情回望这段人生，让玩家被触动

### 评分系统

- 结局得分 = totalScore / maxScore（百分比）
- 4 档评语：≥85% 知己 / ≥60% 相通 / ≥30% 不同但理解 / <30% 截然不同但不批判
- 3 条结语：按得分匹配名句
- 评语必须包含人物名，引用具体事迹
