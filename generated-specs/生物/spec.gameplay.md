# 生物互动模拟学习游戏玩法确认稿

## 1. 玩法结构名称

**层级缩放生命系统模拟器 + 参数实验 + 多题型诊断反馈**

本产品是一个可填充的生物学习成品框架。后续实现管线会读取随附 `skills/内容文件`，从任意生物相关内容中提取生命结构、机制过程、变量关系、实验结论、图谱材料和题目依据，填充为最终用户可直接学习和游玩的互动产品。

核心体验不是静态课件，也不是普通题库，而是：

```text
分子 → 细胞 → 组织/器官 → 个体稳态 → 遗传发育 → 种群/生态系统
```

用户在不同尺度之间缩放，观察生命系统如何由结构产生功能、由变量影响过程、由局部机制影响整体结果。

## 2. 输出模式和设计粒度

| 项目 | 确认值 |
|------|--------|
| 项目名称 | 生物 |
| 输出模式 | fixed-content |
| 设计粒度 | 可填充玩法框架 |
| 内容范围 | 任何生物相关内容 |
| 目标用户 | 多年龄层通用，可由内容文件填充为小学科普、初高中生物、成人通识或综合学习版本 |
| 准确性模式 | strict |
| 技术交付 | 纯前端单文件 HTML + CSS + JS + 内嵌 SVG/Canvas |
| 进度存储 | localStorage |
| 图片缓存 | IndexedDB |

`fixed-content` 表示最终用户打开的是已经填充好的生物互动学习游戏，不需要上传文件、输入内容或操作生成器。内容提取和填充发生在构建阶段。

## 3. 核心循环

每个知识点都必须绑定一个生命机制可视化场景，并走完四阶段闭环：

1. **结构 / 机制讲解**
   - 使用同一个 SVG / Canvas 场景播放生命结构或机制过程。
   - 讲解文字与结构高亮、分子运动、信号传递、器官变化、生态关系同步。
   - 每段讲解 60-120 字，复杂知识点可拆 2-3 段。

2. **参数实验探索**
   - 玩家通过滑块、拖拽、开关、时间轴、图层切换或配对操作调整生命变量。
   - 变量变化必须实时影响结构、曲线、粒子、流动、比例或网络状态。
   - 探索阶段没有对错，目标是建立机制直觉。

3. **多题型答题 / 诊断**
   - 系统锁定某个生命状态、隐藏部分标注，或给出异常现象。
   - 玩家完成选择、点选、拖拽标注、过程排序、机制匹配、曲线判读、遗传推算、生态网络预测、异常诊断等任务。
   - 题目必须基于前两个阶段的同一可视化场景。

4. **同场景反馈演示**
   - 不管答对答错，都用同一机制图动态演示。
   - 答对：高亮关键结构/路径/变量，播放正确机制，给 30-80 字解释。
   - 答错：先演示错误判断会导致什么，再演示正确机制，给 50-100 字温和纠偏。
   - 答错可重试，但首次作答结果仍用于成就判断。

## 4. 内容文件 / skills 映射

后续管线中的完整内容文件通常位于项目根目录 `skills/` 文件夹。构建本产品时必须读取每个文件全文，而不是只看标题、索引或摘要。

| 内容提取维度 | 映射到的产品模块 |
|--------------|------------------|
| 生命结构：分子、细胞器、细胞、组织、器官、系统、个体、种群、生态节点 | SVG 结构图、可点击对象、图层、标注、章节主题 |
| 功能关系：结构-功能、反馈调节、信号传递、能量流、物质循环、遗传关系 | 机制路径、图层匹配题、诊断依据、反馈演示 |
| 生命过程：扩散、渗透、酶促反应、光合作用、呼吸作用、免疫、神经传导、遗传、演替等 | 动态 SVG/Canvas 动画、时间轴状态、参数实验 |
| 生物规则：遗传定律、稳态调节、酶活性规律、生态平衡、能量传递效率等 | 参数响应规则、题目答案、自检项 |
| 实验材料：变量、对照组、实验现象、曲线、数据表、结论 | 参数面板、曲线图、实验诊断题、数据判读 |
| 可操作变量：温度、pH、浓度、光照、氧气、激素、基因型、种群数量等 | 滑块、开关、时间轴、预测题、模拟反馈 |
| 可验证事实：结构名称、功能、代谢路径、遗传概率、生态关系、实验结论 | strict 准确性验收、题目正确答案、解析依据 |
| 视觉主题：显微染色、医学图谱、生态场景、实验室氛围 | UI 气质、背景图 prompt、粒子类型、章节过渡 |

## 5. 章节和内容规模

默认规模沿用物理 / 化学 / 地理互动学习规格：

- 5-7 章。
- 每章 3-4 个知识点。
- 每个知识点 1 道主问题，复杂知识点可增加 1 道进阶题。
- 全书总题量约 20-25 题。
- 每章末尾 1 个知识总结。
- 每章 3 个成就：全对成就、全错调侃成就、探索创意成就。

默认章节可按生命尺度递进：

```text
分子与生化反应 → 细胞结构与运输 → 器官系统与稳态 → 遗传与发育 → 种群与生态系统 → 综合生命系统
```

实际章节可由内容文件决定，例如植物生理、人体系统、细胞生物、遗传专题、生态专题、实验探究等。

## 6. 参数范围和交互变量

本框架需要支持较大范围的动态参数调整，实际参数由 `skills/内容文件` 按知识点填充。

推荐参数池：

| 参数类型 | 示例参数 | 可视化响应 |
|----------|----------|------------|
| 分子 / 生化 | 分子浓度、底物浓度、酶浓度、温度、pH、反应时间 | 粒子碰撞、酶活性曲线、反应速率、产物数量 |
| 细胞 | 膜通透性、通道蛋白数量、外界浓度、细胞内外渗透压、ATP 供应 | 分子跨膜、细胞膨胀/皱缩、主动运输、囊泡移动 |
| 植物生理 | 光照强度、CO₂ 浓度、水分、叶绿素、气孔开度、温度 | 光合作用速率、产氧量、蒸腾、叶片状态 |
| 人体稳态 | 心率、呼吸频率、血氧、血糖、胰岛素、体温、激素水平 | 器官系统响应、负反馈曲线、运输流动 |
| 神经 / 免疫 | 神经信号强度、刺激频率、病原体数量、抗体、免疫细胞数量 | 信号传播、突触传递、免疫清除过程 |
| 遗传 | 亲本基因型、显隐性关系、连锁/独立、突变率、样本数量 | Punnett 方格、表型比例、家系图、概率条 |
| 生态 | 种群数量、捕食压力、资源量、环境变化、迁入迁出、出生率/死亡率 | 食物网、种群曲线、能量流、生态稳定性 |

交互控件包括滑块、双向滑块、图层开关、时间轴、结构点选、分子拖拽、基因型选择器、曲线游标、生态网络节点调节、播放/暂停和重置。

## 7. 题型设计

题型要服务生命机制理解、结构识别、实验分析和诊断判断，默认包含：

| 题型 | 玩法 | 必须依赖的可视化 |
|------|------|------------------|
| 3 选 1 | 根据机制图或实验现象判断正确结论 | 当前结构图 / 曲线 / 模拟状态 |
| 结构点选 | 在细胞、器官、系统、食物网中点选正确结构或节点 | SVG 热区 |
| 拖拽标注 | 把细胞器、器官、反应物、信号、生态角色拖到正确位置 | SVG 标注锚点 |
| 过程排序 | 排列生命过程阶段，如 DNA 转录翻译、免疫应答、生态演替 | SVG 阶段卡片 / 时间轴 |
| 机制匹配 | 将结构与功能、变量与结果、激素与作用、捕食者与猎物匹配 | SVG 网络 / 配对面板 |
| 曲线判读 | 根据酶活性、光合作用、血糖、种群数量等曲线答题 | SVG 曲线图 |
| 遗传推算 | 根据亲本基因型、家系图或表型比例计算结果 | Punnett 方格 / 家系图 |
| 生态网络预测 | 调整某种群或资源后预测食物网变化 | 生态网络图 + 种群曲线 |
| 异常诊断 | 根据异常现象判断可能的机制问题，如缺氧、激素失衡、酶失活 | 机制图 + 状态面板 |

所有题型必须提供可视化反馈，不允许只显示“正确 / 错误”。

## 8. 高精度 SVG / Canvas 可视化方向

SVG 是本产品的核心资产，不是装饰。最终 spec 需要把 SVG 要求写到高优先级约束中。

### SVG 必须承担的内容

- 细胞剖面：细胞膜、细胞核、线粒体、叶绿体、内质网、高尔基体、核糖体、液泡等。
- 细胞膜运输：通道蛋白、载体蛋白、主动运输泵、囊泡、浓度梯度。
- 分子机制图：酶-底物结合、DNA → RNA → 蛋白质、信号通路。
- 组织 / 器官系统：呼吸、循环、消化、神经、内分泌、泌尿、免疫。
- 植物结构：叶片横切、气孔、维管束、根尖、花结构。
- 遗传图谱：Punnett 方格、染色体、基因位点、家系图、比例条。
- 生态网络：食物链、食物网、能量金字塔、物质循环、种群曲线。
- 实验图表：对照组、变量、曲线、数据点、误差范围。

### SVG 质量要求

- 每个知识点至少有一个 SVG 或 SVG + Canvas 混合的核心可视化。
- 同一知识点四阶段必须共用同一个可视化，通过 `vis_state` 切换状态。
- 结构图必须有清晰图层、图例、标注、方向或流程箭头。
- 复杂结构要分层管理，避免混成一张不可维护的图。
- 关键生命变量必须可视编码：颜色、线宽、透明度、粒子密度、流速、曲线高度、节点大小等都要有对应含义。
- 标注要有避让规则，不能遮挡关键结构；动态标注需要平滑移动。
- 可交互对象要有 hover / active / selected / locked / correct / incorrect 状态。
- 移动端 SVG 必须可缩放、可平移或重排，核心标注不能被裁切。

### Canvas 适用范围

Canvas 用于大量粒子或连续动态过程：

- 分子随机运动、扩散、渗透。
- 红细胞流动、氧气/二氧化碳运输。
- 神经信号传播、电位波。
- 病原体扩散、免疫细胞追踪和吞噬。
- 生态种群粒子、能量流、物质循环。
- 花粉传播、种子扩散、细胞分裂动画。

SVG 与 Canvas 可以混用：SVG 负责结构、路径、标注、坐标、图例、交互热区；Canvas 负责粒子、流动、扩散和连续动态。

## 9. 页面 / 状态结构

```text
首次打开
└── 图片缓存检查
    ├── cache hit：载入背景图 → 标题页
    └── cache miss：资产准备页 → 生成背景图 → 写入 IndexedDB → 标题页

标题页
├── 开始学习
└── 继续学习

章节主页
├── 已完成章节：可回看
├── 当前章节：高亮可进入
└── 未解锁章节：锁定

章节学习
├── 章节过渡
├── 知识点闭环 × 3-4
│   ├── 结构 / 机制讲解
│   ├── 参数实验探索
│   ├── 多题型答题 / 诊断
│   └── 同场景反馈演示
└── 章末总结

通关页
├── 成就墙
├── 各章一句话回顾
└── 重新开始 / 回看章节
```

## 10. 生图资产范围和时机

本产品需要真实 AI 生图，但仅用于标题页 / 全局背景图，核心生物结构图、机制图、遗传图、器官系统图、生态网络和实验图表必须由 SVG / Canvas / HTML 实现，不能用位图替代。

| 项目 | 确认值 |
|------|--------|
| 是否需要真实位图生图 | 需要 |
| 生图资产范围 | 标题页 / 全局背景图 |
| 核心生物可视化 | SVG / Canvas，不使用 AI 图片替代 |
| IMAGE_GENERATION_TIMING | first-run-cache |
| 缓存方式 | 首次打开生成缺失背景图，写入 IndexedDB，之后只读缓存 |
| build-time 例外 | 无 |

背景图方向由内容文件主题自适应。默认提示词方向：

```text
immersive life science laboratory, holographic biological systems, microscopic fluorescence, anatomical atlas panels, ecological network data screens, precise scientific visualization, cinematic lighting, no people, no readable text, wide background composition
```

最终 spec 需要保留完整生图契约：`IMAGE_ASSET_MANIFEST`、`IMAGE_ASSET_RUNTIME_STATE`、IndexedDB Blob 缓存记录、prompt hash、cache key、生成锁、失败重试、cache hit 不调用生图 API。

## 11. 视觉方向

视觉气质：**沉浸式生命科学实验室 + 显微荧光 + 精密解剖图谱 + 生态数据舱**。

视觉不要锁死为单一蓝绿实验室风，应根据内容文件和章节尺度自适应：

- 分子 / 细胞：显微荧光、染色切片、半透明膜结构、粒子运动。
- 植物生理：叶绿体、光照、维管束、水分流动、清新自然色。
- 人体系统：医学图谱、器官剖面、血流/气体交换、稳态仪表。
- 神经 / 免疫：电信号、突触、病原体、免疫细胞、警戒反馈。
- 遗传：清晰谱系图、染色体、基因位点、概率格。
- 生态：自然能量流、食物网、种群曲线、物质循环。

UI 需要酷炫，但不能牺牲生物结构识别和机制判断：

- 文字对比度必须达标。
- 生物编码色彩不能为了风格而混淆结构、功能或变量含义。
- 配色要多样，但同一核心可视化最多突出 2-3 个强色系，其余结构降饱和或提高透明度。
- 题目状态和探索状态要视觉区分。
- `prefers-reduced-motion` 下减少粒子和过渡，但保留核心机制演示。

## 12. 准确性模式

准确性模式为 `strict`。

必须严格处理：

- 结构名称、结构位置、结构功能。
- 代谢路径、生理调节、实验变量、实验结论。
- 遗传概率、基因型 / 表型关系、家系图判断。
- 生态关系、能量流、物质循环、种群变化。
- 曲线图、数据表、题目答案、错误选项和反馈解析。

允许创作发挥的范围：

- UI 氛围。
- 成就名称和描述。
- 鼓励语、章节过渡文案。
- 非事实性的任务包装。
- 背景图美术风格。

不得为了酷炫视觉改错生命机制，不得凭空编造核心结构、真实实验结论、遗传规律或生态关系。
